# 核心架构总览

本章节从整体上介绍 Goal-Web 的核心架构组件，帮助你形成完整的心智模型，然后再分别深入到各个子章节。

## 你将在这里学到什么

- 应用是如何启动、如何优雅关闭的
- 服务容器如何进行依赖注入和对象管理
- 服务提供者如何把不同模块“挂载”到应用中
- 门面（Facades）如何在保持可测试性的前提下简化调用

1. [应用生命周期](./application-lifecycle.md)
2. [服务容器](./service-container.md)
3. [服务提供者](./providers.md)
4. [门面（Facades）](./facades.md)
